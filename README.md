# DeepSeek 对话导出工具

将 DeepSeek 导出的 JSON 格式对话记录转换为美观、可读的 HTML 文件。

## 🚀 快速开始

### 1. 获取 DeepSeek 对话数据

1. 访问 [DeepSeek 聊天页面](https://chat.deepseek.com/)
2. 点击左下角菜单 → **选项** → **数据管理**
3. 点击 **导出所有历史对话**
4. 下载生成的 `conversation.json` 文件

### 2. 使用导出工具

1. 下载 `index.html` 文件并打开，或打开[这个网站](https://eniline.github.io/DeepSeek_conversation_2_html/)
2. 上传下载的 `conversation.json` 文件
3. （可选）设置日期范围和深度思考选项
4. 勾选需要导出的对话
5. 点击 **📥 一键导出 HTML** 按钮

## ✨ 功能特性

- 📁 **简单上传**：支持点击或拖拽上传 JSON 文件
- 🔍 **日期筛选**：按日期范围筛选要导出的对话
- 🧠 **深度思考控制**：可选择是否包含 AI 的深度思考过程
- ✅ **灵活选择**：支持全选/单选特定对话进行导出
- 🎨 **精美排版**：生成的 HTML 文件具有清晰的样式和结构
- 📑 **目录导航**：自动生成交互式目录，方便快速定位
- 📱 **响应式设计**：适配桌面和移动设备
